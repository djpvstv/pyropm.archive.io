PyroPM Version 1.0 Released!


With the first release of PyroPM, a few months of effort comes to a (temporary) end. And with a release comes a lot of features, including:

Brand New Menus - Beautiful redesigned menus make their debut with custom UI throughout.
Condensed Stagelist with New Stages - Duke it out on a select few balanced stages with a streamlined stage select.
Build Updater - New update? No problem - Download the update directly from the launcher.

And if you thought I was done...

New and Returning Characters - Characters from past Smash games and newcomers join in with the first release of PyroPMEX, an addon pack to the main build.

Download the build from the builds page or from Brawl Vault. You can download the main build as well as the PyroPMEX addon pack, or if you choose you can download the PyroPMEX standalone build. Each standalone build has options for Wii and Dolphin, and the addon build only contains files that you must install yourself. Read the installation guide under the builds page for further information.
